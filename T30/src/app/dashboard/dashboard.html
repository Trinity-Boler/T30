CleanPlay: Greener Gaming for a Sustainable Future

CleanPlay

CleanPlay is an innovation launched in early 2025 that brings clean energy awareness directly to the gaming community. The platform enables players to offset the power their consoles use by subscribing to renewable energy sources such as wind and solar.

It integrates with gaming systems like the PlayStation 5, rewarding players with exclusive in-game items, achievements, and digital collectibles for supporting clean power. This turns gaming from being purely energy-consuming into a more environmentally conscious and participatory experience.

CleanPlay partners with energy providers and developers to verify renewable offsets, ensuring that every gaming session contributes to a cleaner and more sustainable energy grid. It represents a new intersection between entertainment technology and environmental responsibility, using gamification to encourage positive change within one of the largest digital industries.

Source: Sony Interactive: Enabling Environmental Action with the CleanPlay App (March 2025)

Technical Overview

The T30 project is built as a Single Page Application (SPA) using Angular for the frontend, Node.js and Express for the backend API, and MongoDB for the database. The site is hosted on NGINX for efficient static file serving and HTTPS configuration.

The architecture follows a modern MERN-like stack but uses Angular for improved modularity and scalability. Environment variables are handled with dotenv, and RESTful endpoints fetch and display data dynamically from MongoDB.

The charts are created with Chart.js, providing interactive visualizations. Chart.js is utilized for rendering interactive charts on the Summary and Report pages, providing visual insights into CleanPlay adoption and energy distribution. User authentication is managed with JWT tokens stored in local storage for session persistence.
